# 🎨 UI/UX 설계서

> **프로젝트명:** Operation Epic Fury — 커피내기 슬롯머신  
> **버전:** v0.1  
> **작성일:** 2026-03-31  

---

## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **즉시 시작** | URL 접속 후 30초 이내에 첫 스핀 가능 |
| **한 화면 집중** | 한 번에 하나의 액션만 요구 (인지 부하 최소화) |
| **재미 우선** | 정보 전달보다 연출·애니메이션 중심 |
| **모바일 퍼스트** | 스마트폰 세로 모드에서 최적 경험 |
| **접근성** | 색각 이상자도 구분 가능한 심볼 디자인 (이모지 + 라벨) |

**인터랙션 명세:**

| [게임 시작] | 참가자 2명 이상일 때만 활성화, 클릭 시 게임 화면 전환 |

## 4. 애니메이션 명세

### 4.1 릴 회전 애니메이션

| 속성 | 값 |
|------|------|
| 유형 | 세로 무한 스크롤 → 감속 정지 |
| 시작 속도 | 20 심볼/초 |
| 감속 커브 | `cubic-bezier(0.25, 0.1, 0.25, 1)` (ease-out) |
| 릴별 정지 시차 | 릴1: 1.0s, 릴2: 1.5s, 릴3: 2.0s (±0.3s 랜덤) |
| 정지 시 바운스 | 0.1초간 위로 살짝 튀었다가 제자리 |

### 4.2 스핀 버튼

| 상태 | 스타일 |
|------|--------|
| 기본 | 골드 그라데이션 배경, 미세한 펄스 애니메이션 |
| 호버 | 크기 1.05x 확대 + 글로우 |
| 클릭 | 0.95x로 줄었다 원래 크기로 (0.1초) |
| 비활성 | 회색 처리 + 로딩 스피너 |

### 4.3 결과 연출

| 단계 | 애니메이션 | 시간 |
|------|----------|------|
| 릴 전체 정지 | — | 0s |
| 조합명 팝업 | fadeIn + scaleUp (0→1) | 0.3s |
| 점수 카운트업 | 0 → 최종 점수 (숫자 롤링) | 0.5s |
| 잭팟 시 | 화면 골드 플래시 + 별 파티클 | 1.0s |
| 즉시 패배 시 | 화면 적색 플래시 + 흔들림 | 0.5s |

### 4.4 최종 결과 연출

| 단계 | 애니메이션 | 시간 |
|------|----------|------|
| 순위 1위 공개 | slideIn from left | 0.5s |
| 순위 2위 공개 | slideIn from left | 0.5s (0.5s 후) |
| ... | ... | ... |
| 최하위 공개 전 | 드럼롤 사운드 + 화면 진동 | 1.5s |
| 패자 공개 | scaleUp + 적색 강조 + 사운드 | 0.5s |
| 컨페티 | Canvas 파티클 낙하 | 3.0s |

## 5. 반응형 레이아웃

### 5.1 브레이크포인트

| 크기 | 범위 | 레이아웃 |
|------|------|---------|
| 모바일 S | ~374px | 릴 크기 축소 (48px 심볼) |
| 모바일 M | 375~479px | 기본 모바일 (64px 심볼) |
| 모바일 L | 480~767px | 여유 있는 모바일 |
| 태블릿 | 768~1023px | 릴 확대 + 사이드 스코어보드 |
| 데스크톱 | 1024px~ | 최대 폭 제한 (800px) 중앙 배치 |

### 5.2 모바일 vs 데스크톱 차이

| 요소 | 모바일 | 데스크톱 |
|------|--------|---------|
| 릴 크기 | 64px 심볼 | 96px 심볼 |
| 스코어보드 | 릴 하단 | 릴 우측 사이드 패널 |
| 스핀 버튼 | 화면 하단 고정 (sticky) | 릴 하단 |
| 참가자 입력 | 풀 와이드 | 중앙 400px 카드 |

## 6. 사운드 디자인

| 이벤트 | 사운드 | 볼륨 |
|--------|--------|------|
| 스핀 시작 | 릴 회전음 (윙~) | 중 |
| 릴 개별 정지 | "딱" 클릭음 | 중 |
| 더블 매치 | 작은 벨 사운드 | 중 |
| 트리플 매치 | 승리 팡파레 (짧은) | 높 |
| 잭팟 | 카지노 잭팟 사운드 + 코인 쏟아지는 소리 | 높 |
| 즉시 패배 | "빠앙~" 실패음 | 중 |
| 서든데스 진입 | 긴장감 있는 배경음 | 낮 |
| 최종 패자 공개 | 드럼롤 → 심벌즈 크래시 | 높 |
| 버튼 클릭 | 가벼운 클릭음 | 낮 |

> 🔇 모든 사운드는 음소거 토글 버튼으로 on/off 가능 (기본: on)

## 7. 접근성 (A11y)

| 항목 | 대응 |
|------|------|
| 키보드 네비게이션 | Tab 순서: 이름 입력 → 추가 → 참가자 리스트 → 게임 시작 → 스핀 |
| 스크린 리더 | 릴 결과를 `aria-live="assertive"`로 알림 |
| 고대비 | 텍스트 대비율 WCAG AA 기준 (4.5:1) 충족 |
| 모션 감소 | `prefers-reduced-motion` 미디어 쿼리 대응 (애니메이션 간소화) |
| 심볼 구분 | 이모지 + 텍스트 라벨 병행 (색각 이상자 대응) |
