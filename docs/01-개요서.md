## 1. 프로젝트 개요

커피내기를 재미있게 결정하기 위한 **슬롯머신형 미니 게임**이다.  
참가자들이 슬롯을 돌려 결과에 따라 커피를 사는 사람(패자) 또는 얻어 마시는 사람(승자)을 결정한다.

### 1.1 목적
- 팀·친구 간 커피내기를 공정하고 재미있게 해결
- 슬롯머신의 긴장감과 연출로 엔터테인먼트 요소 제공

### 1.2 대상 사용자
- 직장 동료, 친구, 스터디 그룹 등 2인 이상의 참가자

## 3. 용어 정의

| 용어 | 정의 |
|------|------|
| **릴 (Reel)** | 슬롯머신에서 세로로 회전하는 심볼 열. 기본 3릴 구성. |
| **심볼 (Symbol)** | 릴 위에 표시되는 개별 아이콘/이미지 (예: ☕, 🫘, 🍩 등). |
| **페이라인 (Payline)** | 당첨 여부를 판정하는 심볼 조합의 가로 라인. |
| **스핀 (Spin)** | 릴을 회전시키는 1회 플레이 동작. |
| **잭팟 (Jackpot)** | 최고 당첨 조합. 해당 참가자는 커피를 무조건 얻어마심. |
| **꽝 (Miss)** | 최저 등급 결과. 트리플 물(💧💧💧) 시 즉시 패배 확정. |
| **커피 셔틀** | 게임 결과 커피를 사야 하는 패자. |
| **참가자 (Player)** | 게임에 참여하는 개별 인원. |
| **라운드 (Round)** | 모든 참가자가 1회씩 스핀을 완료하는 단위. |
| **베팅 (Bet)** | 해당 라운드에서 걸리는 커피의 종류/수량. |
| **보상 등급** | 당첨 조합에 따라 부여되는 혜택 수준 (면제, 선택권 등). |
| **RNG (Random Number Generator)** | 릴 결과를 결정하는 난수 생성 로직. |

## 4. 게임 규칙

### 4.1 기본 규칙
1. **참가자 등록:** 2명 이상의 참가자 이름을 입력한다.
2. **라운드 진행:** 참가자별로 순서대로 스핀 버튼을 누른다.
3. **결과 판정:** 각 참가자의 스핀 결과(심볼 조합)에 따라 점수를 산출한다.
4. **패자 결정:** 라운드 종료 후 가장 낮은 점수를 받은 참가자가 커피를 산다.
5. **동점 처리:** 최하위 동점 시 해당 참가자들끼리 추가 스핀(서든데스)을 진행한다.

### 4.2 선택 규칙 (옵션)
- **더블 오어 낫싱:** 잭팟 당첨 시 커피 2잔 면제 or 벌칙 2배 도박 가능
- **리스핀:** 1라운드에 1회, 결과가 마음에 안 들면 다시 돌릴 수 있음 (단, 두 번째 결과를 무조건 수용)

## 7. 게임 흐름

```
[시작 화면]
     │
     ▼
[참가자 이름 입력] ── 2명 이상
     │
     ▼
[라운드 시작]
     │
     ▼
[참가자 N 스핀] ──▶ 릴 회전 애니메이션 ──▶ 결과 표시
     │                                         │
     │         (다음 참가자 있음)                │
     ◄─────────────────────────────────────────┘
     │
     ▼ (모든 참가자 완료)
[결과 집계 & 순위 표시]
     │
     ├── 동점 시 ──▶ [서든데스 스핀]
     │
     ▼
[최종 결과 발표]
  "☕ OOO님이 커피를 삽니다!"
     │
     ▼
[다시 하기 / 종료]
```

## 8. 화면 구성

### 8.1 시작 화면
- 게임 타이틀 로고
- 참가자 이름 입력 폼 (동적 추가/삭제)
- "게임 시작" 버튼

### 8.2 게임 화면
- 3릴 슬롯머신 UI (중앙 배치)
- 현재 참가자 이름 표시
- 스핀 버튼 (레버 애니메이션 또는 버튼)
- 결과 심볼 + 점수 표시
- 참가자별 현재 점수 사이드 패널

### 8.3 결과 화면
- 전체 참가자 순위 테이블
- 패자 강조 연출 (애니메이션 + 사운드)
- "커피 셔틀: OOO" 대형 텍스트
- "다시 하기" / "공유하기" 버튼
