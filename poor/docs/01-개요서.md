## 1. 프로젝트 개요

소비 습관을 입력하면 미래의 파산 시점을 예언하고 조리돌림해 주는 **팩폭형 엔터테인먼트 웹 서비스**이다.  
유저가 최근 가장 쓸데없이 돈 쓴 곳 3개를 텍스트로 입력하면, LLM API가 분석하여 파산 시점과 파산 후의 처참한 모습을 묘사한다.

### 1.1 목적
- 쓸데없는 소비 습관을 유머러스하게 팩폭하여 자조적 웃음 유발
- "나도 해봐" 심리를 이용한 바이럴 공유 극대화
- 소비 습관에 대한 가벼운 반성의 계기 제공 (하지만 결국 웃음)

### 1.2 대상 사용자
- 2030 세대 SNS 활발 이용자
- 충동 소비를 자조적으로 즐기는 MZ세대
- 친구들과 결과를 공유하며 놀고 싶은 사람들

## 2. 서비스 컨셉

| 항목 | 내용 |
|------|------|
| 장르 | 팩폭형 소비 분석 엔터테인먼트 |
| 핵심 재미 | 예상치 못한 처참한 미래 묘사, 자조적 웃음 |
| 플랫폼 | 모바일 웹 (반응형) |
| 테마 | 형광 네온 + 다크 배경, 도파민 폭발 컬러 |
| 기술 핵심 | LLM API 연동 (가벼운 프롬프트 기반) |
| 공유 방식 | 결과 이미지(1080×1920) 다운로드 + 링크 공유 |

---

## 3. 용어 정의

| 용어 | 정의 |
|------|------|
| **팩폭** | 팩트(사실)로 폭행. 현실 직시를 강제하는 유머 방식 |
| **디지털 거지** | 무분별한 소비로 인해 디지털 세상에서조차 거지가 된 상태 |
| **파산 예언** | LLM이 생성하는 유저의 미래 파산 시나리오 |
| **소비 고백** | 유저가 입력하는 쓸데없는 소비 내역 3가지 |
| **조리돌림** | AI가 유저의 소비를 신랄하게 비꼬는 코멘트 |
| **사이버 각설이** | 파산 후 온라인에서 구걸하는 가상의 미래 모습 |
| **결과 카드** | 파산 예언 결과를 1080×1920 이미지로 렌더링한 공유용 카드 |
| **거지력** | 유저의 파산 가능성을 수치화한 지표 (0~100%) |

## 4. 작동 방식

### 4.1 전체 흐름

```
[닉네임 입력] → [쓸데없는 소비 3가지 입력] → [AI 분석 중 (로딩 연출)]
                                                      │
                                               [LLM API 호출]
                                                      │
                                               [파산 예언 결과 생성]
                                                      │
                                            ┌─────────┴──────────┐
                                            │                    │
                                     [결과 화면 표시]     [결과 카드 생성]
                                            │                    │
                                     [공유 버튼]          [이미지 다운로드]
                                            │
                                     [다시 하기]
```

### 4.2 입력 예시

| 입력 항목 | 예시 |
|-----------|------|
| 닉네임 | 이현민 |
| 소비 1 | 탕후루 3만 원어치 |
| 소비 2 | 안 입는 요가복 |
| 소비 3 | 쓸데없는 구독 서비스 |

### 4.3 출력 예시

> "이현민 님은 이딴 식의 소비를 유지할 경우 **2026년 11월경** 한강 대교 밑에서 무료 와이파이를 구걸하는 **'사이버 각설이'**로 전락할 확률이 **98%**입니다.  
> 탕후루에 3만 원을 쓰면서 '나를 위한 소비'라고 합리화하고 있죠? 그 탕후루가 녹는 속도보다 님의 통장 잔고가 녹는 속도가 더 빠릅니다."

## 5. 도파민 포인트

| 포인트 | 설명 |
|--------|------|
| **예상 밖의 처참함** | 평범한 소비에 대한 극단적 파산 시나리오로 충격과 웃음 |
| **개인화된 팩폭** | 닉네임 + 실제 입력한 소비 내역을 반영한 맞춤형 조롱 |
| **구체적 파산 시점** | "2026년 11월"처럼 구체적 날짜를 제시해 현실감 부여 |
| **거지력 수치화** | 98%, 87% 등 수치로 파산 확률을 표현해 비교 욕구 자극 |
| **공유 욕구 자극** | "나 한 달 뒤에 파산한대 ㅋㅋㅋ" 자조적 자랑 |

## 6. 바이럴 전략

### 6.1 공유 메커니즘
- **결과 이미지 다운로드:** 1080×1920 비율의 세로형 결과 카드 → 인스타 스토리, 카톡 최적화
- **링크 공유:** "나도 해봐" 멘트와 함께 링크 복사 원클릭
- **Open Graph 태그:** 링크 공유 시 미리보기에 "○○님은 X월에 파산합니다" 노출

### 6.2 바이럴 심리 활용
- 자신의 "하찮은 소비"를 은근히 자랑하고 싶은 심리
- 친구의 결과와 비교하고 싶은 경쟁 심리
- "내가 이렇게 거지래 ㅋㅋ" 자조적 유머 공유 욕구

## 7. 기술 스택 (예정)

| 레이어 | 기술 | 선택 이유 |
|--------|------|-----------|
| 마크업 | HTML5 | 표준 웹 |

| 공유 | Open Graph + Clipboard API | 링크 미리보기 + 복사 |
