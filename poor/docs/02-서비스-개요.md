## 1. 서비스 소개

**디지털 거지 시뮬레이터**는 유저의 쓸데없는 소비 습관을 입력받아 AI가 파산 시점을 예언하고 처참한 미래를 묘사해주는 **팩폭형 바이럴 웹 서비스**이다.  
가벼운 텍스트 입력만으로 개인화된 조롱 결과를 생성하고, 1080×1920 이미지 카드로 다운로드·공유할 수 있다.

### 1.1 한 줄 요약
> "쓸데없는 소비 3개 입력하면 AI가 님의 파산 날짜를 예언해 드립니다 🪦"

### 1.2 서비스 유형

| 항목 | 내용 |
|------|------|
| 서비스 형태 | 웹 애플리케이션 (SPA) |
| 접근 방식 | URL 접속 즉시 이용 (설치 불필요) |
| 비용 | 무료 |
| 계정 | 불필요 (비로그인) |
| 데이터 저장 | 없음 (세션 한정, 서버 미저장) |
| AI 연동 | LLM API (팩폭 텍스트 생성) |

## 2. 서비스 목표

### 2.1 단기 목표 (v1.0)
- 소비 3가지 입력 → AI 팩폭 → 결과 카드 생성 파이프라인 완성
- 모바일 최적화 UI (형광 네온 테마)
- 결과 이미지(1080×1920) 다운로드 기능
- 링크 복사 + Open Graph 공유 지원
- GitHub Pages 배포

### 2.2 중기 목표 (v1.5~v2.0)
- 거지 등급 시스템 (SSS~F)
- 소비 카테고리별 분석 심화
- 결과 히스토리 로컬 저장
- 친구 대결 모드

### 2.3 장기 목표
- 커뮤니티 피드 (최고의 팩폭 결과 공유)
- 시즌별 테마 (연말 결산, 명절 특집 등)
- 다국어 지원

## 3. 타겟 사용자

### 3.1 주요 타겟

| 사용자 그룹 | 특성 | 사용 상황 |
|-------------|------|-----------|
| MZ 직장인 | 월급 탕진 후 자조 | 월급날 직후, "이번 달도 거지" 시점 |
| 대학생 | 용돈 소진, flex 문화 | 불필요한 소비 후 자기 반성(?) |
| SNS 활발 이용자 | 밈·콘텐츠 소비 & 공유 | 인스타/틱톡/카톡 공유용 |
| 커플/친구 그룹 | 서로 결과 비교 | "너 나보다 더 거지야" 배틀 |

### 3.2 사용자 페르소나

**페르소나 A — 박소비 (27세, 직장인)**
- 월급 300만 원, 매달 27일이면 잔고 0원
- 탕후루, 카페 라떼, 넷플릭스+웨이브+디즈니+ 구독 삼총사
- "돈 어디 갔지?" 가 말버릇
- 결과 나오면 인스타 스토리에 무조건 올림

**페르소나 B — 김무지출 (24세, 대학생)**
- "무지출 챌린지" 도전 3시간 만에 실패
- 편의점 야식, 가챠 뽑기, 안 쓰는 앱 구독이 주 지출
- 친구들이랑 결과 비교하다가 2시간 녹음
- "나 내년에 파산한대 ㅋㅋ" 라고 카톡 단톡방에 공유

## 4. 사용 시나리오

### 4.1 기본 시나리오

```
박소비, 점심시간에 동료와 대화 중 "나 이번 달 또 탕진했어"
   │
   ├─ 동료: "야 이거 해봐 ㅋㅋ" → 링크 공유
   │
   ├─ 박소비: 링크 클릭 → 닉네임 입력 → 소비 3개 입력
   │   - "스타벅스 한 달 15만원"
   │   - "안 보는 넷플릭스 구독"
   │   - "충동 구매한 에어프라이어"
   │
   ├─ AI 분석 중... (로딩 연출: 통장 잔고 줄어드는 애니메이션)
   │
   ├─ 결과: "박소비 님은 2026년 9월경 편의점 삼각김밥으로 
   │         연명하는 '디지털 노숙인'이 될 확률 94%입니다"
   │
   ├─ 박소비: 폭소 → 이미지 다운로드 → 인스타 스토리 업로드
   │
   └─ 다른 동료들: "나도 해봐야지" → 연쇄 바이럴
```

### 4.2 공유 시나리오

```
김무지출, 결과 확인 후 링크 복사
   │
   ├─ 카카오톡 단톡방에 공유
   │   → 미리보기: "김무지출 님의 파산 예정일: 2026년 8월"
   │
   ├─ 친구 3명 클릭 → 각각 소비 입력 → 결과 비교
   │
   └─ "누가 더 거지인가" 배틀 시작 → 추가 바이럴
```

## 5. 서비스 핵심 가치

| 가치 | 설명 |
|------|------|
| **즉각적 재미** | 입력 30초 → 결과 확인 → 폭소. 전체 경험 1분 이내 |
| **개인화된 팩폭** | 내 닉네임, 내 소비 내역이 반영된 맞춤형 조롱 |
| **공유 본능 자극** | "하찮은 소비"를 자랑하고 싶은 심리 활용 |
| **경쟁 유발** | 거지력 수치화로 친구 간 비교 욕구 극대화 |
| **리플레이 가치** | 다른 소비를 입력하면 다른 결과 → 반복 플레이 |

## 6. 경쟁 서비스 분석

| 서비스 | 특징 | 한계 |
|--------|------|------|
| MBTI 테스트류 | 선택지 기반, 결과 유형 고정 | 개인화 없음, 매번 같은 결과 |
| 관상/사주 앱 | 사진/생년월일 기반 | AI 텍스트 분석 아님 |
| 심리 테스트 | 다수 문항 → 결과 | 시간 소요 큼, 텍스트 입력 아님 |
| 소비 분석 앱 (뱅크샐러드 등) | 실제 데이터 기반 분석 | 진지함, 재미 요소 없음 |

---

## 7. 차별화 전략

| 전략 | 설명 |
|------|------|
| **자유 텍스트 입력** | 선택지가 아닌 자유 입력 → 무한한 결과 변형 |
| **LLM 기반 생성** | 매번 다른 팩폭 텍스트 → 리플레이 가치 |
| **도파민 디자인** | 형광 네온 + 과장된 연출로 중독성 UI |
| **이미지 공유 최적화** | 1080×1920 세로형 → 인스타 스토리/카톡 최적화 |
| **30초 완료** | 입력 3줄 → 결과 즉시. 이탈률 최소화 |

## 8. 서비스 운영 방식

| 항목 | 내용 |
|------|------|
| 호스팅 | GitHub Pages (무료 정적 호스팅) |
| API 비용 | LLM API 호출 비용 (API Key는 서버리스 프록시 경유) |
| 모니터링 | Google Analytics 4 |
| 업데이트 | Git push → 자동 배포 |
| 개인정보 | 수집 없음. 입력 데이터는 API 호출 후 즉시 폐기 |
