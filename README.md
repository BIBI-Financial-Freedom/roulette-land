# 🎡 룰렛 랜드

사무실 내기를 공정하고 재미있게 결정하기 위해 만든 웹 게임 플랫폼입니다.
별도 설치 없이 브라우저에서 바로 실행됩니다.

---

## 🎮 게임 목록

### 🎰 원강랜드 — 커피내기 슬롯머신
참가자들이 슬롯머신을 돌려 커피값을 낼 사람을 결정합니다.
꼴등 혹은 1등이 커피를 사는 규칙 중 선택할 수 있습니다.

### 🔫 러시안 룰렛 — 꽝 뽑기
누가 벌칙을 받을지 러시안 룰렛 방식으로 결정합니다.
5가지 모드(클래식 / 페어 / 하드 / 데스매치 / 서바이벌)와
카운트다운 타이머, 결과 공유 기능을 제공합니다.

---

## 📁 구조

```
hub/                  # 게임 선택 메인 허브
coffee/               # 원강랜드 (커피 슬롯머신)
russian-roulette/     # 러시안 룰렛 (꽝 뽑기)
```

## 🚀 실행 방법

`hub/index.html` 을 브라우저에서 열면 됩니다.
