## 1. 서비스 소개

**문해력 바닥 인증기**는 유저가 입력한 텍스트를 AI가 분석하여 **정신적 어휘 연령**을 판별하고 처참하게 조롱해 주는 **어휘력 팩폭 테스트 서비스**이다.  
"최근에 한 변명"이나 "친구에게 돈 빌려달라고 할 때 쓸 카톡"을 3문장 이상 입력하면, AI가 어휘 다양성·클리셰 의존도·표현력 등을 분석한다.

### 1.1 한 줄 요약
> "님의 어휘력은 7세 수준입니다 — 진짜, 완전, 대박 없이는 말을 못 하시네요 📖"

### 1.2 서비스 유형

| 항목 | 내용 |
|------|------|
| 서비스 형태 | 웹 애플리케이션 (SPA) |
| 접근 방식 | URL 접속 즉시 이용 (설치 불필요) |
| 비용 | 무료 |
| 계정 | 불필요 (비로그인) |
| 데이터 저장 | 없음 (세션 한정, 서버 미저장) |
| AI 연동 | LLM API (텍스트 분석 + 팩폭 생성) |
| 공유 특화 | OG 태그 파라미터로 카카오톡 미리보기 어그로 |

## 2. 서비스 목표

### 2.1 단기 목표 (v1.0)
- 텍스트 입력 → AI 어휘 분석 → 결과 카드 생성 파이프라인 완성
- 모바일 최적화 UI (형광 네온 테마)
- 결과 이미지(1080×1920) 다운로드 기능
- OG 태그 파라미터 기반 카카오톡 미리보기 어그로
- GitHub Pages 배포

### 2.2 중기 목표 (v1.5~v2.0)
- 어휘 연령 등급 시스템 (SSS~F)
- 워드 클라우드 시각화
- 결과 히스토리 로컬 저장
- 서버사이드 OG 태그 렌더링 (동적 미리보기 이미지)

### 2.3 장기 목표
- 대결 모드 (두 유저 텍스트 비교)
- 시대별 어휘 트렌드 분석
- 영어/일본어 등 다국어 지원

## 3. 타겟 사용자

### 3.1 주요 타겟

| 사용자 그룹 | 특성 | 사용 상황 |
|-------------|------|-----------|
| MZ 직장인 | 카톡·슬랙 과다 사용 | 문자 능력에 자신 있다고 착각 중 |
| 대학생 | 과제·리포트 마감 전사 | 어휘력 테스트 겸 놀이 |
| SNS 헤비유저 | 밈·유행어 중독 | 결과 공유로 인스타 스토리 채우기 |
| 커플/친구 그룹 | 서로 텍스트 비교 | "너는 몇 살 나와?" 배틀 |

### 3.2 사용자 페르소나

**페르소나 A — 정진짜 (25세, 대학생)**
- 일상 대화에서 "진짜", "개", "ㄹㅇ"를 남발
- 과제 피드백에 "어휘가 단조롭다"는 교수 코멘트를 받은 적 있음
- 결과 나오면 "아 진짜 7세래 ㅋㅋㅋ" 하면서 단톡방에 공유
- 친구들이 전부 해볼 때까지 안 멈춤

**페르소나 B — 이표현 (29세, 직장인)**
- 본인이 글을 잘 쓴다고 자부
- 테스트 해보고 "17세" 나와서 충격
- "나 다시 해볼게, 이번엔 제대로" → 3번 리플레이
- 최종 결과 22세 나와서 그나마 만족, 스토리에 올림

## 4. 사용 시나리오

### 4.1 기본 시나리오

```
정진짜, 점심시간에 스마트폰으로 링크 클릭
   │
   ├─ 친구: "야 이거 해봐 ㅋㅋ 나 7세 나옴" → 카톡 링크 공유
   │         카톡 미리보기: "정진짜 님의 어휘력은 ?세 수준입니다"
   │
   ├─ 정진짜: 링크 클릭 → 닉네임 입력 → 주제 선택(변명)
   │
   ├─ 텍스트 입력:
   │   "아 진짜 이거 제 잘못 아니에요 완전 억울한 건데
   │    아무도 안 믿어줌 대박 짜증 나는데 솔직히 저는
   │    되게 잘한 거거든요 그냥 운이 없었을 뿐이에요"
   │
   ├─ AI 분석 중... (로딩: 사전 애니메이션)
   │
   ├─ 결과: "정진짜 님의 어휘 연령: 7세
   │         '진짜'를 3번, '완전'을 1번, '대박'을 1번 사용.
   │         님의 어휘 창고에는 5개의 단어만 있습니다."
   │
   ├─ 정진짜: 폭소 → 이미지 다운로드 → 카톡 단톡방 공유
   │
   └─ 단톡방 전원 클릭 → 연쇄 바이럴
```

### 4.2 OG 태그 어그로 시나리오

```
이표현, 결과 확인 후 공유 버튼 클릭
   │
   ├─ URL 생성: example.com/voca/?name=이표현&age=17&keyword=표현력부족
   │
   ├─ 카카오톡에 URL 붙여넣기
   │   → 자동 미리보기:
   │   ┌──────────────────────────────┐
   │   │ 📖 문해력 바닥 인증기         │
   │   │ 이표현 님의 어휘력은           │
   │   │ 17세 수준입니다               │
   │   └──────────────────────────────┘
   │
   ├─ 친구들: "17세? ㅋㅋ 나는 몇 살 나올까" → 클릭
   │
   └─ 연쇄 바이럴 발동
```

## 5. 서비스 핵심 가치

| 가치 | 설명 |
|------|------|
| **즉각적 결과** | 텍스트 입력 1분 → 결과 확인. 전체 경험 2분 이내 |
| **개인화된 조롱** | 내가 실제로 쓴 단어를 콕 집어 분석 |
| **OG 어그로** | 링크만 보내도 결과가 미리보기에 뜸 → 클릭 불가피 |
| **비교 경쟁** | 어휘 연령 수치화로 친구 간 비교 |
| **리플레이 가치** | "이번엔 제대로 써볼게!" → 반복 시도 |

## 6. 경쟁 서비스 분석

| 서비스 | 특징 | 한계 |
|--------|------|------|
| 맞춤법 검사기 (부산대 등) | 맞춤법 교정 중심 | 재미 요소 없음, 어휘력 미분석 |
| MBTI 테스트류 | 선택지 기반 | 텍스트 분석 아님, 매번 같은 유형 |
| 국어 능력 시험 | 정식 시험 형태 | 재미없고 시간 오래 걸림 |
| 어휘력 퀴즈 앱 | 문제 풀기 형식 | AI 분석 아님, 바이럴 요소 부족 |

---

## 7. 차별화 전략

| 전략 | 설명 |
|------|------|
| **자유 텍스트 분석** | 보기 선택이 아닌, 실제 작문을 AI가 분석 → 진짜 어휘력 |
| **LLM 기반 개인화** | 매번 다른 텍스트 → 매번 다른 분석 결과 |
| **OG 태그 어그로** | 카카오톡 미리보기에 결과 노출 → 타 서비스 대비 압도적 CTR |
| **도파민 디자인** | 형광 네온 컬러로 자극적 UI |
| **이미지 공유 최적화** | 1080×1920 세로형 → 인스타 스토리 최적 |

## 8. 서비스 운영 방식

| 항목 | 내용 |
|------|------|
| 호스팅 | GitHub Pages (무료 정적 호스팅) |
| API 비용 | LLM API 호출 비용 (서버리스 프록시 경유) |
| OG 태그 | 클라이언트 URL 파라미터 기반 (서버 렌더링 불필요) |
| 모니터링 | Google Analytics 4 |
| 업데이트 | Git push → 자동 배포 |
| 개인정보 | 수집 없음. 입력 텍스트는 API 호출 후 즉시 폐기 |
