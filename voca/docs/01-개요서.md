## 1. 프로젝트 개요

유저가 입력한 텍스트 몇 줄을 AI가 분석하여 **"정신적 어휘 연령"**을 판별하고 처참하게 조롱해 주는 **어휘력 테스트 엔터테인먼트 웹 서비스**이다.  
변명이나 카톡 멘트를 3문장 이상 입력하면, LLM이 어휘 다양성, 표현력, 클리셰 의존도 등을 분석해 사용자의 어휘 나이를 진단한다.

### 1.1 목적
- 유저의 어휘력을 유머러스하게 평가하여 자조적 웃음 유발
- "내 어휘력은 7세래 ㅋㅋ" 결과 공유를 통한 바이럴 극대화
- 카카오톡 OG 미리보기를 활용한 어그로 공유 전략

### 1.2 대상 사용자
- 2030 세대 SNS 활발 이용자
- 카카오톡·인스타그램을 주로 사용하는 MZ세대
- 친구들과 결과를 비교하며 놀고 싶은 사람들

## 2. 서비스 컨셉

| 항목 | 내용 |
|------|------|
| 장르 | 어휘력 분석 팩폭형 엔터테인먼트 |
| 핵심 재미 | 예상치 못한 낮은 어휘 연령 + 신랄한 분석 코멘트 |
| 플랫폼 | 모바일 웹 (반응형) |

| 기술 핵심 | LLM API 연동 (텍스트 분석 기반) |
| 공유 방식 | 결과 이미지(1080×1920) 다운로드 + OG 태그 어그로 링크 |

## 3. 용어 정의

| 용어 | 정의 |
|------|------|
| **어휘 연령** | AI가 판별한 유저의 정신적 어휘 수준을 나이로 환산한 지표 |
| **어휘 빈곤** | 제한된 어휘만 반복 사용하는 상태. "진짜", "완전", "대박" 무한루프 |
| **클리셰 의존도** | 진부한 표현이나 유행어에 의존하는 정도 |
| **문해력 점수** | 0~100점 기준 어휘력 수치화 지표 |
| **팩폭 리포트** | AI가 제공하는 어휘 분석 + 조롱 코멘트 |
| **결과 카드** | 어휘 연령 결과를 1080×1920 이미지로 렌더링한 공유용 카드 |
| **OG 어그로** | URL 파라미터로 점수를 전달해 카카오톡 미리보기에 결과가 노출되는 전략 |
| **텍스트 고백** | 유저가 입력하는 3문장 이상의 분석 대상 텍스트 |

## 4. 작동 방식

### 4.1 전체 흐름

```
[닉네임 입력] → [주제 선택 (변명 / 돈 빌리기)] → [텍스트 3문장+ 입력]
                                                           │
                                                    [AI 분석 중 (로딩)]
                                                           │
                                                    [LLM API 호출]
                                                           │
                                                    [어휘 분석 결과 생성]
                                                           │
                                                  ┌────────┴─────────┐
                                                  │                  │
                                           [결과 화면 표시]    [결과 카드 생성]
                                                  │                  │
                                           [공유 버튼]        [이미지 다운로드]
                                                  │
                                           [다시 하기]
```

### 4.2 입력 예시

**주제 A — 최근에 한 변명**
> "아 진짜 이거 제 잘못이 아니라 완전 상황이 그런 거예요. 대박 억울한 건데 아무도 안 믿어주잖아요. 솔직히 저는 되게 잘한 건데 운이 없었던 거라고요."

**주제 B — 친구에게 돈 빌려달라고 할 때 쓸 카톡**
> "야 나 진짜 급한데 좀만 빌려줘 ㅠㅠ 다음 달에 무조건 갚을게 진짜로. 지금 완전 빈털터리라서 밥도 못 먹고 있어 죽겠어."

### 4.3 출력 예시

> "님 어휘력은 **2010년대에 머물러** 있네요.  
> **'진짜', '완전', '대박'** 없이는 말을 못 하는 **7세 지능 수준**입니다.  
> 3문장 안에 '진짜'를 3번 쓰신 거 알고 계시죠? 어휘력이 아니라 **복사-붙여넣기 수준**입니다."

## 5. 도파민 포인트

| 포인트 | 설명 |
|--------|------|
| **참을 수 없는 굴욕감** | "7세 수준"이라는 직격탄에 자존심 자극 → 공유 |
| **개인화된 분석** | 내가 실제로 쓴 단어를 콕 집어 조롱 |
| **비교 욕구** | "난 12세인데 넌?" 친구 간 어휘 연령 배틀 |
| **OG 태그 어그로** | 링크만 보내도 "○○님의 어휘력은 7세 수준" 이 뜸 |
| **재시도 욕구** | "이번엔 제대로 써볼게!" → 리플레이 |

## 6. 바이럴 전략

### 6.1 핵심 바이럴 기믹 — OG 태그 어그로

URL 끝에 파라미터로 점수와 평가 키워드를 넘겨서, 카카오톡 미리보기에 결과가 그대로 노출되는 전략.

```
https://example.com/voca/?name=이현민&age=7&keyword=복사붙여넣기
```

**카카오톡 미리보기 표시:**
```
┌────────────────────────────────┐
│  📖 문해력 바닥 인증기          │
│                                │
│  이현민 님의 어휘력은            │
│  7세 수준입니다                 │
│                                │
│  키워드: #복사붙여넣기           │
└────────────────────────────────┘
```

→ 클릭을 안 할 수가 없는 어그로

### 6.2 공유 메커니즘
- **결과 이미지 다운로드:** 1080×1920 세로형 → 인스타 스토리/카톡 최적화
- **링크 복사:** OG 파라미터 포함 URL 자동 생성
- **Web Share API:** 모바일 네이티브 공유 시트

### 6.3 바이럴 심리 활용
- "나 7세래 ㅋㅋ" 자조적 어그로
- "너는 몇 살 나와?" 비교 유도
- "이번엔 진지하게 써봤는데도 12세래" 리플레이 유도

## 7. 기술 스택 (예정)

| 레이어 | 기술 | 선택 이유 |
|--------|------|-----------|
| 마크업 | HTML5 | 표준 웹 |
| 스타일 | CSS3 | 형광 네온 테마, 애니메이션 |
| 스크립트 | Vanilla JavaScript (ES2022+) | 경량, 빌드리스 |
| LLM API | OpenAI / Gemini API | 어휘 분석 + 팩폭 생성 |
| 이미지 생성 | html2canvas | 결과 카드 캡처 |
| OG 태그 | 동적 meta 태그 + URL 파라미터 | 카카오톡 미리보기 어그로 |
| 배포 | GitHub Pages | 무료 정적 호스팅 |
