# 🔫 러시안 룰렛 — 꽝 뽑기

> 모임에서 벌칙 대상을 **스릴 넘치고 공정하게** 결정하는 웹 기반 파티 게임

## 🎮 게임 소개

리볼버 탄창(6칸)을 모티브로 한 UI에서 참가자들이 돌아가며 방아쇠를 당기고,
꽝(탄환)에 걸린 사람이 벌칙을 수행합니다.

- **인원:** 2~6명
- **조작:** 터치/클릭 한 번으로 플레이
- **설치:** 불필요 — 브라우저에서 즉시 실행

## 🕹️ 게임 모드

| 모드 | 탄환 | 특징 | 버전 |
|------|------|------|------|
| 🎯 클래식 | 1발 | 순차 진행, 기본 룰 | v1.0 |
| ⚖️ 페어 | 1발 | 매 턴 탄창 리스핀 (모두 동일 확률) | v1.0 |
| 🔥 하드 | 2발 | 높은 위험, 빠른 진행 | v1.5 |
| ☠️ 데스매치 | 3발 | 50:50 생존율, 극한 긴장감 | v1.5 |

## 📸 화면 구성

1. **시작 화면** — 참가자 등록, 모드 선택, 벌칙 설정
2. **장전 화면** — 탄창 스핀 애니메이션
3. **게임 화면** — 방아쇠 당기기 + 실시간 확률 표시
4. **결과 화면** — 꽝 당첨자 & 벌칙 공개

## ⚙️ 기술 스택

- **HTML5 + CSS3 + Vanilla JavaScript** (프레임워크 없음)
- **RNG:** `crypto.getRandomValues()` (암호학적 안전 난수)
- **사운드:** Web Audio API (외부 파일 불필요)
- **애니메이션:** CSS Keyframes (heartbeat, screen-shake, flash)
- **반응형:** 모바일 퍼스트 설계 (374px ~ 1024px+)

## 🚀 실행 방법

`src/index.html` 파일을 브라우저에서 직접 열면 됩니다.
서버 설정이 필요 없습니다 (`file://` 프로토콜 지원).
